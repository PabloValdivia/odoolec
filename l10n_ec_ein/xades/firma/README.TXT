Se debe copiar el jar conjuntamnete con las librerias que se encuentran en
la carpeta /lib

Para ejecutar el proyecto desde la línea de comandos, escriba lo siguiente:

java -jar firmaXadesBes.jar arg1 arg2 arg3

Donde los argumentos son los siguientes:

* arg1: xml en formato cadena.
* arg2: ubicación de la firma electrónica encriptada en base64
* arg3: clave de la firma electrónica encriptada en base64
